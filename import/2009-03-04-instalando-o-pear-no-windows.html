Instalando o PEAR no Windows

Mar 4th, 2009 | Comments

Depois de instalar o interpretador PHP no meu computador só pra poder usar o Doctrine a partir da linha de comando, fui instalar o PEAR para poder testar algumas coisas na máquina local.

Como a minha instalação padrão do PHP não tinha o PEAR, fui me aventurar. Utilizei o go-pear, um auto-instalador que baixa os pacotes básicos para você ter sua instalação do PEAR rodando.

1) Baixe o arquivo http://pear.php.net/go-pear com o nome go-pear.php. Recomendo utilizar o seu diretório de instalação do PHP (C:\Arquivos de Programas\PHP).

2) Abra o prompt de comando (Iniciar -> Executar -> “cmd”) e vá até o diretório onde você salvou o arquivo go-pear.php (cd “C:\Arquivos de Programas\PHP”, no nosso exemplo)

3) Execute o arquivo com o comando php -q go-pear.php e sigaos passos da instalação (geralmente é só ir apertando <Enter>).

4) No final ele criará um arquivo chamado PEAR_ENV.reg no diretório atual. Em seguida é só executá-lo para finalizar a instalação (só dar um duplo clique).

Agora você pode utilizar todos os comandos do pacote PEAR, simplesmente digitando pear <comando>.

Por exemplo, com um pear list você terá as informações sobre os pacotes do PEAR instalados atualmente.
