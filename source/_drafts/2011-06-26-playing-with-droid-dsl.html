In order to play with the Droid DSL source code, you have to:

  1. Install Eclipse. Helios SR2 (version 3.6.2) release is recommended.
  2. Open Eclipse, go to Help > Install new Software
  3. Select the Helios update site (http://download.eclipse.org/releases/helios)
  4. Under the Modeling category select the following packages: "Acceleo SDK" (v. 3.0.2), "EMF - Eclipse Modeling Framework SDK" (v. 2.6.1) and "Xtext SDK" (v. 1.0.2). Install them all and restart the Eclipse.
  5. Clone the github repository at 
To play with the generated plugins, you need to:
  1. Build the plugins
  2. File > Export
  3. Plug-in Development > Deployable plug-ins and fragments
  4. Select all the plugins and choose a directory
